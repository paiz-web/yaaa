Apa Itu CSS?

CSS (Cascading Style Sheets) adalah bahasa desain yang digunakan untuk mengatur tampilan dan tata letak halaman web. CSS memungkinkan kita untuk memisahkan konten HTML dari presentasi visual.

Cara Menyisipkan CSS

  • Inline: Langsung dalam atribut elemen HTML.
  • Internal: Di dalam tag <style> di bagian <head> HTML.
  • Eksternal: Menggunakan file .css yang terpisah.
<p style="color: green;">Teks ini hijau dengan inline CSS</p>

Selector CSS

Selector digunakan untuk memilih elemen HTML yang ingin diberi gaya.

  • element: Memilih semua elemen tertentu, contohnya </p>
  • .class: Memilih elemen dengan class tertentu
  • #id: Memilih elemen dengan ID tertentu
p {
  color: blue;
}

.highlight {
  background-color: yellow;
}

#utama {
  font-size: 20px;
}

Mengatur Warna

CSS dapat mengatur warna teks, latar belakang, dan border:

h1 {
  color: green;
  background-color: #e0ffe0;
  border: 1px solid green;
}

Contoh Judul Berwarna

Mengatur Teks

Beberapa properti teks yang umum digunakan:

  • font-size: Ukuran huruf
  • font-weight: Ketebalan huruf
  • text-align: Perataan teks
  • color: Warna teks
.judul {
  font-size: 24px;
  font-weight: bold;
  text-align: center;
  color: #00963c;
}

Contoh Judul Teks

Box Model

Model kotak (box model) adalah dasar tata letak di CSS:

  • Content: Isi elemen
  • Padding: Jarak antara isi dan border
  • Border: Garis luar elemen
  • Margin: Jarak luar antara elemen lainnya
.kotak {
  margin: 20px;
  padding: 15px;
  border: 2px solid #00963c;
  background-color: #e9fbee;
}
Ini adalah contoh dari Box Model.
