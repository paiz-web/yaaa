Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif, seperti membuat tombol yang bisa diklik, popup, validasi form, dan lainnya.

Variabel

Variabel digunakan untuk menyimpan data. Di JavaScript kita bisa menggunakan let, const, atau var.

let nama = "Budi";
const umur = 20;
var kota = "Jakarta";

console.log(nama);

Lihat hasilnya di konsol dengan menekan F12 > Console.

Fungsi

Fungsi adalah blok kode yang bisa digunakan ulang. Contoh:

function sapa(nama) {
  return "Halo, " + nama;
}

console.log(sapa("Ani"));

Buka konsol untuk melihat hasilnya.

Manipulasi DOM

DOM (Document Object Model) memungkinkan kita mengubah isi HTML lewat JavaScript.

document.getElementById("demo").innerHTML = "Halo dari JavaScript!";

Teks ini akan diubah.

Event Listener

Event listener digunakan untuk menjalankan kode saat sebuah aksi dilakukan, seperti klik tombol, arahkan mouse, dan lainnya.

document.getElementById("tombol").addEventListener("click", function() {
  alert("Tombol diklik!");
});

Array

Array digunakan untuk menyimpan banyak data dalam satu variabel.

let buah = ["apel", "jeruk", "mangga"];
console.log(buah[1]); // jeruk

Buka konsol (F12) untuk melihat hasil dari console.log.

Perulangan

Perulangan digunakan untuk menjalankan kode secara berulang. Contohnya perulangan for:

for (let i = 1; i <= 3; i++) {
  console.log("Perulangan ke-" + i);
}

Lihat hasilnya di konsol dengan menekan F12 > Console.
