1. Struktur Dasar HTML

Tag ini adalah struktur HTML yang harus ada saat menggunakan HTML.

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

Hello, World!

2. Heading

Tag HTML yang digunakan untuk membuat heading atau judul dalam sebuah halaman web.

<h1>Judul Besar</h1>
<h2>Subjudul</h2>

Judul Besar

Subjudul

3. Paragraf

Tag <p> dalam HTML digunakan untuk membuat paragraf.

<p>Ini adalah sebuah paragraf.</p>

Ini adalah sebuah paragraf.

4. Tautan

Tautan digunakan untuk menghubungkan halaman satu ke halaman lain, baik di dalam website yang sama maupun ke website lain.

<a href="https://segacode.com">Kunjungi SeGaCODE</a>
Kunjungi SeGaCODE

5. Gambar

Tag <img> untuk menampilkan gambar yang kita inginkan. 

<img src="https://via.placeholder.com/150" alt="Contoh Gambar">

6. Daftar

<ul> untuk membuat daftar tidak berurutan (bullet point).

<ol> digunakan untuk membuat daftar berurutan (dengan angka atau huruf).

Di dalam kedua elemen tersebut, kita menggunakan <li> (list item) untuk mendefinisikan setiap item dalam daftar.

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<ol>
  <li>Langkah Pertama</li>
  <li>Langkah Kedua</li>
</ol>

  • CSS

  2. Langkah Kedua

7. Tabel

<table> Merupakan elemen utama yang digunakan untuk membuat sebuah tabel.

<tr> (table row) Digunakan untuk membuat satu baris dalam tabel. Setiap baris bisa berisi beberapa kolom.

<th> (table header) Digunakan untuk membuat kolom judul di baris pertama tabel. Biasanya tampil tebal dan rata tengah, untuk menunjukkan bahwa itu adalah nama/label dari kolom.

<table>
  <tr><th>Nama</th><th>Umur</th></tr>
  <tr><td>Budi</td><td>20</td></tr>
</table>
